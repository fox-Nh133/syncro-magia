# MagiaML 仕様書

MagiaML は **Magia Studio** 上でライトデバイスの発光演出をプログラムするための軽量 DSL（ドメイン固有言語）です。音楽に合わせたキーフレームごとにコードを記述し、60台のデバイス（5行 × 12列）を制御します。

---

## 基本構文

```
セレクタ { プロパティ: 値; プロパティ: 値; }
```

- 1つのキーフレームに複数のルールを記述可能（後のルールが優先して上書き）
- プロパティの区切りは `;`（末尾の `;` は省略可）
- 1ルールにつきセレクタは1つ

---

## セレクタ

| セレクタ | 対象デバイス |
|---|---|
| `all` | 全60台 |
| `range(lo%, hi%)` | インデックス位置が `lo%`〜`hi%` の範囲に入るデバイス |
| `filter(odd)` | 奇数番デバイス（index 0, 2, 4, … — 1番・3番・5番…） |
| `filter(even)` | 偶数番デバイス（index 1, 3, 5, …） |
| `index(n)` | 0始まりで n 番目の1台 |

### `range()` の計算方法

デバイスの位置 `pos = index / 59`（0.0〜1.0）が `lo/100 ≤ pos ≤ hi/100` を満たすデバイスが対象。

---

## プロパティ

| プロパティ | 型 | デフォルト | 説明 |
|---|---|---|---|
| `color` | CSS カラー（hex） | `#ffffff` | 発光色 |
| `effect` | 文字列 | `static` | 発光エフェクト |
| `duration` | 数値（秒） | `1` | エフェクトの継続時間 |

---

## エフェクト一覧

### `static`
常時点灯。キーフレームの時刻以降、次のキーフレームで上書きされるまで点灯し続ける。

### `flash`
キーフレーム開始から **min(duration, 0.3) 秒間** 点灯し、その後消灯。

### `wave`
左から右へ波が走るエフェクト。  
- デバイス `i` は経過時間が `arriveAt = (i / 59) × duration` に達したとき点灯
- 点灯は **0.4 秒間** 継続した後消灯

### `split`
中央から外側へ光が広がるエフェクト。  
- 中央からの距離 `distNorm = |i - 29.5| / 29.5`（0=中央, 1=端）
- デバイス `i` は経過時間が `arriveAt = distNorm × duration` に達したとき点灯し、その後**点灯し続ける**

### `split-flash`
`split` と同様に中央から外側へ広がるが、各デバイスは **0.3 秒間フラッシュ**して消灯。  
- 点灯タイミング: `arriveAt = distNorm × duration`
- 点灯時間: 0.3 秒

---

## キーフレームの重ね合わせ

複数のキーフレームが時系列順に積み重なり、各デバイスの状態は**最後に適用されたルールで決定**される。

```
現在時刻以前の全キーフレーム → 時刻順に順次適用 → 後が上書き
```

---

## 記述例

```
/* 全デバイスを消灯 */
all { color: #000000; effect: static; }

/* 前半50%のデバイスを紫のウェーブで 2 秒かけて点灯 */
range(0%, 50%) { color: #7c3aed; effect: wave; duration: 2; }

/* 奇数番デバイスをピンクで 0.5 秒フラッシュ */
filter(odd) { color: #ec4899; effect: flash; duration: 0.5; }

/* インデックス 0 のデバイスのみ白で点灯 */
index(0) { color: #ffffff; effect: static; }

/* 全デバイスを中央から 1.5 秒でスプリット点灯 */
all { color: #a78bfa; effect: split; duration: 1.5; }

/* 全デバイスを中央から広がるスプリットフラッシュ */
all { color: #ec4899; effect: split-flash; duration: 2; }
```

## JSON エクスポート形式

Magia Studio からエクスポートされる JSON の構造：

```json
{
  "audioName": "filename.mp3",
  "keyframes": [
    {
      "time": 4.139,
      "code": "all { color: #ec4899; effect: wave; duration: 0.5; }"
    }
  ]
}
```

| フィールド | 型 | 説明 |
|---|---|---|
| `audioName` | string | 読み込んだ音源ファイル名 |
| `keyframes[].time` | number（秒） | キーフレームのタイムスタンプ（小数点3桁） |
| `keyframes[].code` | string | そのキーフレームの MagiaML コード |

---

## 仕様上の制約・注意事項

- デバイス数は固定 **60台**（変更不可）
- 1ルールのセレクタは1つのみ。`range(...) filter(...)` のような複合セレクタは**サポートされていない**（パーサーは最後のセレクタのみ認識する）
- `color` は CSS hex カラー形式（例: `#7c3aed`）を推奨
- `duration: 0` 以下の値は内部で `1` として扱われる（`wave`, `split`, `split-flash`）
- `flash` の実際の点灯時間は `min(duration, 0.3)` 秒
